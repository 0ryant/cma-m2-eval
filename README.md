# cma-m2-eval

Evaluation suite for the Controller Mind Architecture (CMA) / M2 policy layer.

Pre-registered empirical test of M2-policy agents against a Latent State Model (LSM) baseline across topology, counterfactual, and social signal battery dimensions.

**Canonical protocol document:** `docs/preregistration.md`
**Verdict contract (frozen 2026-03-11):** `docs/verdict_contract.md`

See `docs/preregistration.md` for observables, thresholds, and OSF registration details before running.

This project was preregistered on OSF:
DOI: 10.17605/OSF.IO/GESYH

The registered protocol is frozen. Repository changes after this point
do not alter the preregistration and must be interpreted relative to
the archived OSF record.
